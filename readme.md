## Setup

1. Clone the repository:
    ```sh
    git clone https://github.com/your-username/simple-website.git
    ```
2. Navigate to the project directory:
    ```sh
    cd simple-website
    ```

## Usage

Open `index.html` in your web browser to view the website.
